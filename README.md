# 📊 Stock Price Prediction Dashboard

## 🔍 Overview
This project predicts stock prices using a machine learning model and visualizes the results using Power BI.

## 🚀 Features
- Predicted vs Actual stock price comparison
- Error analysis over time
- Model performance evaluation using MAE
- Interactive dashboard with date filter

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Jupyter Notebook
- Power BI

## 📈 Key Insights
- Model achieved MAE ≈ 1.11
- Predictions closely follow actual stock trends
- Error remains low and stable over time

## 📂 Files
- `stock_prediction.ipynb` → Model building
- `df_stocks.csv` → Dataset
- `Stock_Price_Dashboard.pbix` → Dashboard
